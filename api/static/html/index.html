API Routes

Welcome to the API for my capstone project. The project I am building is an e-commerce website with a points system, where users are also allowed to sell products. The points system operates in a way that the more points a user accumulates, the higher their rank becomes. As they ascend through the ranks, they gain access to discounts and deals. Please see the routes provided by this API below:

Route Method Description
Users
/users GET Fetch All Users' data
/user/:id GET Fetch a single user's data
/register POST Register a user (takes a payload: firstName, lastName, gender, email, userPass and userProfile)
/login POST Log in (requires password and email)
/user/:id PATCH Edit a user's information (also takes a payload)
/user/:id DELETE Ban a user
Products
/products GET Get all products
/product/:prodID GET Get a single product's data
/product/:seller_id POST Add a product. "seller_id" is the user that is selling the product
/product/:prodID PATCH Edit a product's details
/product/:prodID DELETE Delete a product
Cart
/cart GET Fetch all the items in the cart
/cart POST Add a product to the cart
/cart/:productID DELETE Delete an item from the cart
Transactions
/record POST Record a transaction, whether it is a buy or sell
/avg-quant/:userID GET Get a specific user's Average based on how much they've sold and bought overtime. This will be the points system used on the website
